11. 配列 Arrays

数理情報研究室 濱田龍義

配列 (Array) とは

  • 変数が増えると管理が大変
  • 配列とは，たくさんの変数を管理する仕組み
  • 順序をつけて管理される
  • インデックス(index) と呼ばれる数字で管理
  • コンピュータはデータを取り扱う機械
  • データ（値）を取り扱う仕組みをデータ構造と呼ぶ
  • 配列は，基本的なデータ構造のひとつ

例

	    
	      int[] years = { 1920, 1972, 1980, 1996, 2010 }
  • int[] データ型
  • years 変数名（配列名）
  • { 1920, 1972, 1980, 1996, 2010 }
  • データ，値，要素

配列の作り方と利用方法

作り方

  1. 宣言
  2. 生成
  3. 代入

利用方法

  1. 伝統的な for文
  2. モダンな for文

（第5回復習）変数の作成

  1. データ型の宣言 (int, float, char, ...)
  2. 名前（変数名）
  3. 値（データ）の代入
		
		  int x; // 1. データ型, 2. 名前
		  x = 12; // 3. 値の代入
		
	      
		
		  int x = 12; // 1. 2. 3.
注意）1. 2. はソースコード中で1度しか宣言できません．
		
		  int x = 12;
		  int x;
		
		Duplicate local variable x
		
コンソールにエラーメッセージ

配列の作り方

1. 配列の宣言

  • int[] x;
  • データ型[] 変数名;

2. 配列の生成

  • x = new int[2];
  • 変数名 = new データ型[要素数];
  • 配列の長さ，すなわち要素数は，あらかじめ決めておく必要がある．

3. 配列の代入

ひとつずつ代入

		
		  int[] x;  // 配列の宣言
		  x = new int[2] // 配列の生成
		  x[0] = 12;  // 配列の0番目に代入
		  x[1] = 2;  // 配列の1番目に代入

同時に宣言，生成

		
		  int[] x = new int [2]

一息に宣言，生成，代入

		
		  int[] x = {12, 2}

draw(){...}の中で配列を作ると，繰り返しで重くなりますので，避けたほうが良いです．

配列は Array クラスで実現

		
		  int[] x = {12, 2}
  • x は Array クラスのオブジェクト
  • その変数名は x
		x.length
  • length は配列 x のフィールド（インスタンス変数）
  • 配列 x の長さ，要素数を表す
		x.length - 1

配列の最後の要素のインデックス

    x[i] = random(-1000, 200);
  }
}
void draw() {
  background(0);
  for (int i = 0; i < x.length; i++) {
    x[i] += 0.5;
    float y = i * 0.4;

  }
}

11_09 マウスを追跡する

	    
int num = 60;
int[] x = new int[num];
int[] y = new int[num];
void setup() {
  size(240, 120);
  noStroke();
}
void draw() {
  background(0);
  // Copy array values from back to front
  for (int i = x.length-1; i > 0; i--) {
    x[i] = x[i-1];
    y[i] = y[i-1];
  }
  x[0] = mouseX; // Set the first element
  y[0] = mouseY; // Set the first element
  for (int i = 0; i < x.length; i++) {
    fill(i * 4);
    ellipse(x[i], y[i], 40, 40);
  }
}

11_10 多数のオブジェクトを管理

	    
JitterBug[] bugs = new JitterBug[33];
void setup() {
  size(240, 120);
  for (int i = 0; i < bugs.length; i++) {
    float x = random(width);
    float y = random(height);
    int r = i + 2;
    bugs[i] = new JitterBug(x, y, r);
  }
}
void draw() {
  for (int i = 0; i < bugs.length; i++) {
    bugs[i].move();
    bugs[i].display();
  }
}

11_12 画像のシーケンス

	    
int numFrames = 12; // The number of frames
PImage[] images = new PImage[numFrames]; // Make the array
int currentFrame = 0;
void setup() {
  size(240, 120);
  for (int i = 0; i < images.length; i++) {
    String imageName = "frame-" + nf(i, 4) + ".png";
    images[i] = loadImage(imageName); // Load each image
  }
  frameRate(24);
}
void draw() {
  image(images[currentFrame], 0, 0);
  currentFrame++; // Next frame
  if (currentFrame >= images.length) {
    currentFrame = 0; // Return to first frame
  }
}
