05_13 offset 基準位置からの距離


float x;
int offset = 10;

void setup() {
  size(240, 120);
  x = width/2;
}

void draw() {
  background(204);
  if (mouseX > x) {
    x += 0.5;
    offset = -10;
  }
  if (mouseX < x) {
    x -= 0.5;
    offset = 10;
  }
  // Draw arrow left or right depending on "offset" value
  line(x, 0, x, height);
  line(mouseX, mouseY, mouseX + offset, mouseY - 10);
  line(mouseX, mouseY, mouseX + offset, mouseY + 10);
  line(mouseX, mouseY, mouseX + offset*3, mouseY);
}

05_14 円の境界


int x = 120;
int y = 60;
int radius = 12;
void setup() {
  size(240, 120);
  ellipseMode(RADIUS);
}
void draw() {
  background(204);
  float d = dist(mouseX, mouseY, x, y);
  if (d < radius) {
    radius++;
    fill(0);
  } else {
    fill(255);
  }
  ellipse(x, y, radius, radius);
}

命題 (Proposition)

「正しい」か「正しくない」かが決まる文や式

例

  • 脂質を摂取することによって，どのような体調の変化があるか調べたい．
  • マウスA群は高脂肪食を与えられた．
  • マウスB群は高脂肪食を与えられなかった．
  • 「高脂肪食を与えられた」という命題を真とすると，
  • 「高脂肪食を与えられなかった」という命題は偽となる．
  • 「正しい」か「正しくない」というのは，主観的な概念ではないことに注意すること．

条件 (condition)

変数 $x$ によって決まる命題 $p(x)$ のこと

  • 「$x$は正の数である」という文章の真偽は，$x$で決まる．
  • $x=2 \Rightarrow $ 真 (true, 1)
  • $x=-1 \Rightarrow $ 偽 (false, 0)
  • 「$x$は正の数である」という文章は変数 $x$によって決まる条件である．

論理回路→デジタル回路

  • AND, OR, NOTを組み合わせた論理回路によって，あらゆるデジタル回路を作成できる
  • デジタル回路とは以下の2通りの回路
    • 電気信号が通ってない→0
    • 電気信号が通っている→1
