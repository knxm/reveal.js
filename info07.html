座標系とは何か？

座標系

  1. 平面内に原点$O$を定める．
  2. ベクトル$\overrightarrow{OA}$が決まる．
  3. $\mathbf{e_{1}}$を定める．$\mathbf{e_{1}}\ne 0$
  4. $\mathbf{e_{1}, e_{2}}$が平行にならないように$\mathbf{e_{2}}$を定める．$\mathbf{e_{2}}\ne 0$
  5. $\overrightarrow{OA}=x\mathbf{e_{1}}+y\mathbf{e_{2}}$と表せる．
  6. $(x, y)$を点$A$の座標　(coordinate)と呼ぶ．
  7. 組$(O, \mathbf{e_{1}, e_{2}})$ を座標系 (coordinate system)と呼ぶ．
  8. 特に$\mathbf{e_{1}, e_{2}}$が直交している場合，$(O, \mathbf{e_{1}, e_{2}})$ を直交座標系　(orthogonal coordinate system)と呼ぶ．

原点$O$は，どこでも良い

  • 原点$O$によって，$(x, y)$の値は異なる．

$\mathbf{e_{1}, e_{2}}$の長さ

  • $\mathbf{e_{1}, e_{2}}$によって，$(x, y)$の値は異なる．

Processingの標準座標系

  • 原点$O$はキャンバスの左上隅，上記の図では大きめに表示しているが，実際には$\mathbf{e_{1}, e_{2}}$の長さは1ピクセル（画素）である．また，$\mathbf{e_{2}}$は左上隅から左下隅方向に取る．

  translate(40, 0);  // Move to next joint
  rotate(angle * 2.0);
  strokeWeight(6);
  line(0, 0, 30, 0);
  translate(30, 0);  // Move to next joint
  rotate(angle * 2.5);
  strokeWeight(3);
  line(0, 0, 20, 0);
  angle += speed * angleDirection;
  if ((angle > QUARTER_PI) || (angle < 0)) {
    angleDirection = -angleDirection;
  }
}

スタック (stack)

  • コンピュータで用いられる基本的なデータ構造の一種
  • 後入れ先出し
  • LIFO: Last in First Out, FILO: First in Last out
  • push: データを記憶領域に保存する．
  • pop: データを記憶領域から取り出す．
  • よく例えられるのは，食堂のお盆
  • 座標系を保存する pushMatrix()
  • 座標系を取り出す popMatrix()
  • 「9. 関数」で活躍します．

まとめ
